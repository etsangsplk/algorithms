# String Compression

Compress a string by indicating the lenght of each segment consisting
of the same character.  Consider only letters a-z and A-Z.  If
compressed string is not smaller than original, return original.  

Example:

**Input.**
```
Bbbmaa
bbboooa
oaabooo
```

**Output.**
```
Bbbmaa
b3o3a1
oaabooo
```
