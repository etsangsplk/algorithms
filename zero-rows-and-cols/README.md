# Zero Rows and Cols

You are given a matrix of integers of size MxN.  For each zero in the
matrix, set to zeroes the corresponding row and column.

Example:

**Input.**
```
3 4
1 2 3 4
5 6 0 8
9 10 11 12
3 4
1 2 3 0
5 6 0 8
9 10 11 12
3 3
1 2 3
0 5 6
7 8 0
```

**Output.**
```
1 2 0 4
0 0 0 0
9 10 0 12
0 0 0 0
0 0 0 0
9 10 0 0
0 2 0
0 0 0
0 0 0
```
